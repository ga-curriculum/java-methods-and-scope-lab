**Learning objective :** By the end of this lesson, you will be able to create java methods that can take in arguments of various data types, perform a task and return the result.

## Requirements

Write code in the <code class="filepath">./src/main/java/org.exercise/Lab.java</code> file to implement the various methods below.

### Exercise 1: `maxOfTwoNumbers()`

Define a method, `maxOfTwoNumbers()`, that takes two numbers as arguments and returns the largest of them.

**Example**: `maxOfTwoNumbers(1, 5)` would return `5`.

### Exercise 2: `maxOfThree()`

Define a method, `maxOfThree()`, that takes three numbers as arguments and returns the largest of them.

**Example**: `maxOfThree(1, 5, 7)` would return `7`.

### Exercise 3: `isCharacterAVowel()`

Write a method, `isCharacterAVowel()`, that takes a *character* (not a string) and returns `true` if it's a vowel and `false` otherwise.

**Examples**:

- `isCharacterAVowel('a')` would return `true`.
- `isCharacterAVowel('A')` would also return `true`.

### Exercise 4: `argsList()`

Define a method, `argsList()`, that returns the number of arguments passed to it when called.

**Example**: `argsList(1, 5, 7)` would return `3`.

### Exercise 5: `reverseString()`

Define a method, `reverseString()`, that computes the reversal of a string.

**Example**: `reverseString("ten tar")` should return the string `"rat net"`.
