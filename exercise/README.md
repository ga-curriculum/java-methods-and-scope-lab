This lab provides an opportunity to practice creating methods.

## Requirements

Write code in the <code class="filepath">./src/main/java/org.exercise/Lab.java</code> file to implement various methods, following the requirements listed below:

- Define a method, `maxOfTwoNumbers()`, that takes two numbers as arguments and returns the largest of them. You should use conditional statements to accomplish this. Do a Google search to figure this out if you forgot how conditionals work.
- Define a method, `maxOfThree()`, that takes three numbers as arguments and returns the largest of them.
- Write a method, `isCharacterAVowel()`, that takes a character (i.e., a string of length `1`) and returns `true` if it's a vowel and `false` otherwise.
- Write a method called `argsList`, that returns the number of arguments passed to it when called.
- Define a method, `reverseString()`, that computes the reversal of a string. For example, `reverseString("jag testar")` should return the string `"ratset gaj"`.
